# Examples

These examples are small regression prompts for checking whether the skill behaves as intended. They are not model answers and should be adapted to the user's manuscript context.

## Recommended Checks

- WRITE should build a section scaffold before drafting.
- REVIEW should cite only text that appears in the prompt.
- DIAGNOSE should map paragraphs to section components and flag missing functions.
- LEARN should teach one concept and include a short practice task.

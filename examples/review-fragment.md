# REVIEW Example: Fragment

## Prompt

Review the writing quality of this paragraph:

> Due to the fact that macrophages have an important role in inflammatory arthritis, a number of studies have made an evaluation of drug delivery strategies. In order to improve the therapeutic effect, dexamethasone prodrug microspheres were prepared and the uptake was assessed. The results clearly showed a significant improvement, which indicates that this may be a useful strategy.

## Expected Skill Behavior

The assistant should:

1. Run the 5-pass audit only on the provided paragraph.
2. Flag clutter such as "Due to the fact that," "a number of," and "In order to."
3. Flag nominalizations such as "made an evaluation of."
4. Note that "clearly showed a significant improvement" may overstate the evidence if no data are provided.
5. Mark Pass 5 as limited or not assessable because no table, figure, sample size, citation list, or numeric values are supplied.

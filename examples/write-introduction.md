# WRITE Example: Introduction

## Prompt

Help me draft an Introduction for a research article.

Topic: macrophage-targeted dexamethasone prodrug microspheres for inflammatory arthritis.

Study aim: evaluate whether lipid-conjugated dexamethasone prodrug microspheres improve macrophage uptake and reduce inflammatory activation in vitro.

Key context:
- Rheumatoid arthritis involves persistent synovial inflammation.
- Macrophages contribute to cytokine production and tissue damage.
- Dexamethasone is anti-inflammatory but systemic exposure can cause side effects.
- Prodrug microspheres may improve local or cell-associated delivery.

Known findings:
- The microspheres were internalized by macrophage-like cells.
- Treated cells showed lower inflammatory marker expression after stimulation.

## Expected Skill Behavior

The assistant should:

1. Identify the target section as Introduction.
2. Build a Component A-D scaffold before drafting.
3. Ask for missing study-specific details if needed, especially exact model, comparator, and quantitative findings.
4. Avoid inventing data, citations, or mechanism claims.

## Introduction Model

4 Components, 11 Sentence Types. Funnel structure: broad → narrow → specific → your work.

```
COMPONENT A — ESTABLISH IMPORTANCE + BACKGROUND (S1–S3)
  S1: Establish importance of the field
      → "X plays a key role in..." / "X has attracted considerable attention..."
  S2: Provide general background
      → "X is known to..." / "It is well established that..."
  S3: Provide specific background (narrowing)
      → "In particular, X has been shown to..." / "Recent studies have focused on..."

COMPONENT B — DESCRIBE PROBLEM AREA / CURRENT RESEARCH (S4–S7)
  S4: Describe the general problem area
      → "However, X remains poorly understood."
  S5: Describe previous research
      → "Smith et al. (2020) demonstrated that..."
  S6: Describe previous research (continued)
      → "Using X approach, Jones et al. found..."
  S7: Describe another aspect or transition to limitations
      → "While X has been investigated, Y has received less attention."

COMPONENT C — IDENTIFY GAP (S8)
  S8: Identify the gap — CRITICAL PIVOT
      → "To date, no study has..." / "Little is known about..."
      → Must directly address the problem described in Component B

COMPONENT D — DESCRIBE PRESENT PAPER (S9–S11)
  S9: State the purpose/objective
      → "In this study, we investigated..."
  S10: Describe the method briefly
      → "We used X to..."
  S11: State the main finding/contribution
      → "Our results show that..."
```

Key rules: Literature is argument, not catalogue — build a case for the gap. The gap is everything. Preview, don't detail.

## Methods Model

6 Generic Components.

```
COMPONENT 1 — OVERVIEW
  → General overview of methodological approach
  → Purpose of the method (why this approach)

COMPONENT 2 — MATERIALS/SUBJECTS/EQUIPMENT
  → Precise descriptions: concentrations, model numbers, demographics
  → Inclusion/exclusion criteria; source of materials

COMPONENT 3 — PROCEDURE/PROTOCOL
  → Step-by-step process description
  → Reference modified protocols: "method of X with minor modifications"

COMPONENT 4 — DATA ANALYSIS/STATISTICS
  → Software name and version; which test for which comparison
  → Significance level stated explicitly

COMPONENT 5 (WOVEN) — JUSTIFICATION OF CHOICES
  → Explain WHY a method was chosen
  → "This method was chosen because..."

COMPONENT 6 (OPTIONAL) — PROBLEMS ENCOUNTERED
  → Describe problems and how they were addressed
  → Only include if it affects interpretation
```

Key rules: Reproducibility is paramount. Use subheadings. Reference established protocols. Passive voice standard; active for framing decisions.

## Results Model

4 Components (Cycling). NOT chronological — rhetorically structured.

```
COMPONENT 1 — CONTEXT/TRANSITION (Opens each subsection)
  → Revisit literature/aim/prediction/hypothesis
  → Revisit/summarise the method
  → General statement about results
  → Invite reader to view graphic + describe content

COMPONENT 2 — SPECIFIC/KEY RESULTS (Core findings)
  → Specific results ± evaluative language
  → Comparison with other studies + "happy words"
  → Comparison with model/simulation/predicted results
  → Explanation of results via known facts

COMPONENT 3 — PROBLEMS/ISSUES (Brief, honest)
  → Problems/issues with results ± reasons
  → Minimize impact, redirect toward positive

COMPONENT 4 — POSSIBLE IMPLICATIONS (Closes subsection)
  → Possible implications + "happy words"
  → Brief preview; full discussion comes later
```

Key rules: Begin with transition material, never raw data. Use evaluative language. Compare with literature. End subsections positively. Tense: Present Simple = general truth; Past Simple = this study. The model cycles.

### Results — Language by Component

| Component | Key Language |
|-----------|-------------|
| Context/Transition (Component 1) | "As predicted, X was found to..." / "Consistent with previous findings (Smith, 2020),..." / "To assess X, we performed Y..." / "As shown in Figure 1,..." / "Overall, X showed a significant increase..." |
| Key Results (Component 2) | "X was significantly higher than Y (p < 0.05)." / "Notably, Z showed the largest effect..." / "Strikingly, X remained unchanged..." / "This finding is consistent with Smith et al. who reported..." / "This can be explained by the fact that..." |
| Problems/Issues (Component 3) | "However, X was not significantly different from Y." / "Unexpectedly, Z showed no change..." / "A limitation of this finding is that..." |
| Implications (Component 4) | "These results suggest that X may be useful for..." / "This finding has important implications for..." / "The observed effect could potentially be applied to..." |

## Discussion Model

9 Components. Interpretation-driven. Introduction-Discussion symmetry (mirror the funnel in reverse).

```
COMPONENT 1 — ANNOUNCE STRUCTURE/CONTENT (Optional)
  → Roadmap: "This section discusses X, Y, and Z."

COMPONENT 2 — STATE ACHIEVEMENT/CONTRIBUTION (Free-flowing)
  → May appear throughout. Achievement = internal; Contribution = outward-facing.

COMPONENT 3 — REVISIT BACKGROUND TO 'REBOOT' READER
  → Selective reminder of relevant background from Introduction.

COMPONENT 4 — REVISIT GAP/AIM/METHOD
  → Restate gap (Introduction C), aim (Introduction D), method — symmetry.

COMPONENT 5 — REVISIT RESULTS AND EXPLORE IMPLICATIONS (Central)
  → Do NOT repeat results — INTERPRET what they mean.
  → Use confidence-calibrated language. Longest component.

COMPONENT 6 — MAP TO LITERATURE/KNOWLEDGE (Woven throughout)
  → "consistent with," "contrary to," "the first time"
  → Literature thread strongest here.

COMPONENT 7 — IDENTIFY LIMITATIONS AND FUTURE WORK
  → Honest constraints, framed as opportunities.
  → Distinguish others' future work vs own planned work (starred forms).

COMPONENT 8 — RESTATE ACHIEVEMENT/CONTRIBUTION/IMPACT
  → Stronger than Component 2. "Very happy words" and strong verbs.

COMPONENT 9 — IDENTIFY POTENTIAL APPLICATIONS
  → Practical implications: who could use this and how.
```

Key rules: Start by "rebooting" reader. Interpret, don't repeat. Calibrate confidence to evidence. Achievement ≠ Contribution. Components 5 and 6 interleaved. Close with contribution + applications.

## Conclusion Model

11 Components. No fixed ordering. Synthesis-driven: delivers take-home message about outcome and impact. Typical length: 100–200 words, 1–2 paragraphs.

```
COMPONENT 1  — WHAT IS IN THE PAPER
  → "This paper has demonstrated..." / "In the present paper,..."

COMPONENT 2  — WHAT THE PAPER/STUDY HAS ACHIEVED
  → "We have synthesized and characterized..."

COMPONENT 3  — RELEVANT BACKGROUND INFORMATION
  → Brief selective reminder of context.

COMPONENT 4  — THE GAP/AIM/NEED FOR THE STUDY
  → Introduction-Conclusion symmetry.

COMPONENT 5  — THE METHOD/APPROACH
  → Brief mention, often combined with achievement.

COMPONENT 6  — KEY RESULTS WITH EVALUATIVE COMMENTS
  → Only most important findings — NOT a full recap.

COMPONENT 7  — IMPLICATIONS OF THE RESULTS
  → "This highlights the potential to..."

COMPONENT 8  — POTENTIAL OR ACTUAL LIMITATIONS
  → Often brief or absent in short Conclusions.

COMPONENT 9  — POTENTIAL OR ACTUAL APPLICATIONS

COMPONENT 10 — HOW THE STUDY ADVANCES KNOWLEDGE
  → "paves the way for..." / "fills key gaps"

COMPONENT 11 — POTENTIAL FUTURE DIRECTIONS FOR RESEARCH
  → Next steps for the field or own planned work.
```

Key rules: No new language patterns — draws from all previous units. Most common components: 2, 6, 7, 9, 10. Own your contribution explicitly.

**Ownership Strategies** — avoid ambiguity about what is YOUR contribution:

| Strategy | Example |
|----------|---------|
| Use Past Simple or Present Perfect | "We **have synthesized**..." |
| Use active rather than passive | "**We** designed and implemented..." |
| Use human grammatical subject | "**Our results** fill key gaps..." |
| Explicit reference to paper/study | "**This study** clearly demonstrates..." |

## Abstract Model

2 Parts, 10 Components. Standalone document. Write it LAST. 80–250 words typically.

```
PART 1 — OPENING
  Component 1: SIGNIFICANCE OF THE TOPIC / ESSENTIAL FACTUAL BACKGROUND
    → Present Simple for established facts
  Component 2: THE CHALLENGE/PROBLEM
    → "challenge," "limited," "little work," "difficulty"
  Component 3: WHAT THE PAPER/STUDY DOES (+ ACHIEVEMENT/VALUE)
    → "This paper presents..." / "We report herein..."

PART 2 — BODY
  Component 4: METHOD/MATERIALS
    → Past Simple
  Component 5: RESULTS (+ COMPARISONS WITH EXISTING RESULTS)
    → Past Simple or Present Simple
  Component 6: IMPLICATIONS
    → "suggest that," "indicate that," "may/could"
  Component 7: MAPPING TO EXISTING KNOWLEDGE
  Component 8: ACHIEVEMENT/VALUE/CONTRIBUTION
    → Present Perfect or Present Simple
  Component 9: APPLICATIONS
  Component 10: FLOW AND NARRATIVE COHERENCE
    → Link components smoothly; combine sentence functions where useful
```

Four abstract types: Simple/Standard, Structured (headings), With Significance/Highlights, Graphical.

Key rules: Prioritize central achievement. Keep sentences ~22 words. Combine sentence functions. Don't overclaim. Ensure narrative flow across all components. Make ownership explicit.

## Title Writing

7-Point Analysis Strategy:

1. **Average length** — typically ~12 words
2. **Acronyms** — minimize; outsiders today = interdisciplinary readers tomorrow
3. **Keywords vs keyword list** — complement, don't repeat; use list to extend reach
4. **Grammar** — minimize compound nouns and prepositions; replace prepositions with content words
5. **Structural content** — A/An opening (new offer), colon (topic:focus or term:definition), method, purpose, "happy words"
6. **Title-paper consistency** — must not overstate; use modals if needed
7. **Working title trap** — use outcome/value title, not research-activity title

## Language Reference Tables

### Reporting Verb Strength Scale

| Strength | Verbs | When to Use |
|----------|-------|-------------|
| **Strong** | show, demonstrate, prove, establish, confirm | Well-supported, widely accepted findings |
| **Moderate** | suggest, indicate, report, find, observe | Supported but context-dependent findings |
| **Tentative** | speculate, hypothesize, propose, postulate | Preliminary or theoretical claims |
| **Critical** | claim, argue, contend, assert | Writer distances self from the claim |

### Hedging Devices

| Function | Devices |
|----------|---------|
| Limit generalization | "tend to," "in most cases," "generally," "it appears that" |
| Indicate uncertainty | "may," "might," "could," "it is possible that" |
| Soften claims | "suggests," "indicates," "is consistent with" |
| Attribute to source | "according to," "as reported by" |

### "Happy Words" — Positive Evaluative Language

| Context | Words/Phrases |
|---------|--------------|
| Agreement | "consistent with," "in agreement with," "confirms," "supports" |
| Significance | "notably," "strikingly," "remarkably," "importantly," "significantly" |
| Positive outcome | "successfully," "effectively," "efficiently," "robustly" |
| Value | "provides valuable insights," "contributes to our understanding" |
| Implication | "has important implications for," "may be useful for" |
| Strength | "clear," "strong," "compelling," "substantial," "marked" |

### Discussion — Comparison Language

| Function | Phrases |
|----------|---------|
| Agreement | "consistent with," "in agreement with," "in accordance with," "in line with" |
| Similarity | "analogous to," "comparable to," "similar," "resemble," "mirror" |
| Difference | "distinct from," "in contrast to," "differ from," "significantly different" |
| Disagreement | "contrary to," "in conflict with," "contradict," "conflict with," "refute" |
| Novelty | "new," "novel," "the first time," "not previously described" |

### Discussion — "Very Happy Words" (!-Substitutes)

compelling, crucial, dramatic, excellent, exceptional, exciting, extraordinary, ideal, invaluable, outstanding, remarkable, striking, superb, undeniable, unique, unprecedented, unquestionably, vital

### Discussion — Speculation Language

| Strength | Structures |
|----------|-----------|
| Moderate hedge | "may/might/could," "it is reasonable to assume that" |
| Strong hedge | "it is conceivable that," "we cannot rule out" |
| Support for speculation | "this is reinforced by," "this is substantiated by," "this points to" |
| Emphatic | "Clearly,...," "Indeed,...," "Perhaps..." |

### Discussion — Future Work Language

| Column A: Need/Urgency | Column B: Direction | Column C: Action |
|------------------------|-------------------|------------------|
| "there is a need for," "further work is needed," "urgent" | "holds promise," "we recommend," "worthwhile" | "should be explored/investigated/replicated/validated" |

Starred forms (writer's own ongoing work): "further work is planned*," "future studies will*," "is being investigated*," "work is underway*," "work is in progress*"

### Discussion — Application Sentence Patterns

- "provides evidence-based guidance for..."
- "is appropriate for..." / "is also suitable for..."
- "could be used to evaluate..."
- "are applicable to other..."
- "will enable..." / "as a broadly applicable approach..."
- "are feasible for widespread use"
- "could eventually lead to recommendations for..."

### Abstract — Language by Component

| Component | Key Language |
|-----------|-------------|
| Significance/Background | "is fundamental to," "is known to," "it is widely accepted that," "key," "central," "widely," "worldwide" |
| Challenge/Problem | "challenge," "limited," "little work," "difficulty," "inadequate," "problem," "risk," "a need for" |
| What Paper Does | "present," "report," "investigate," "propose," "introduce," "describe," "show," "examine" |
| Method/Materials | "are/were analysed," "are/were measured," "are/were evaluated," "are/were used" |
| Results | "demonstrate," "reveal," "exhibit," "increase," "decrease," "result in," "are/was found/observed" |
| Achievement/Value | "novel," "robust," "efficient," "(the) first," "superior," "improved," "significant," "reliable" |
| Implications | "suggest that," "indicate that," "may/might/could," "potentially," "appear to" |
| Applications | "suitable for/in," "potential," "enable," "implement," "wide range of," "relevant for" |

### Sequence Language (Methods — 8 Groups)

| Group | Function | Connectors |
|-------|----------|-----------|
| 1 | Sequential | "First,... Then,... Next,... Finally,..." |
| 2 | Simultaneous | "Simultaneously,... Concurrently,... At the same time,..." |
| 3 | Time reference | "Prior to X,... After X,... Following X,..." |
| 4 | Repetition/duration | "This procedure was repeated N times..." |
| 5 | Condition-dependent | "If/When X occurred, Y was performed..." |
| 6 | Reference to previous | "As described above,... As outlined in Section X,..." |
| 7 | Modification from standard | "With minor modifications from Smith et al.,..." |
| 8 | Completion/transition | "Once X was complete, Y was initiated..." |

### Tense Selection Guide

| Function | Tense | Example |
|----------|-------|---------|
| Established facts | Present Simple | "X plays a key role in..." |
| Describing graphics | Present Simple | "Figure 1 shows that..." |
| Previous research (actions) | Past Simple | "Smith et al. (2020) demonstrated..." |
| Methods performed | Past Simple | "Samples were incubated at 37°C." |
| Specific findings | Past Simple | "X was significantly higher than Y." |
| Recent/ongoing work | Present Perfect | "Previous studies have shown..." |
| Gap identification | Present Perfect (negative) | "No study has investigated..." |

### Preposition Patterns (Methods)

| Pattern | Example |
|---------|---------|
| at + temperature/time | "at 37°C" / "at 15-minute intervals" |
| in + medium/container | "in PBS" / "in a water bath" |
| for + duration/purpose | "for 24 hours" / "for the detection of X" |
| with + instrument/reagent | "with a spectrophotometer" |
| by + method/agent | "by centrifugation" / "by the method of Smith et al." |
| from + source | "from Sigma-Aldrich" |
| to + endpoint | "to a final concentration of" |

### Confidence Calibration

Language strength must match result power:

| Result Strength | Language Level | Examples |
|----------------|---------------|---------|
| Strong, replicated | Assertive | "demonstrate," "confirm," "establish" |
| Supported but not conclusive | Moderate | "suggest," "indicate," "are consistent with" |
| Preliminary or indirect | Tentative | "may," "might," "could," "it is conceivable that" |
| Negative or unexpected | Careful | "unexpectedly," "surprisingly," "we cannot rule out" |

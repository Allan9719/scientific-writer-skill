# Public Release Checklist

Use this checklist before publishing the repository.

## Include

- `SKILL.md`
- `README.md`
- `ATTRIBUTION.md`
- `LICENSE`
- `CHANGELOG.md`
- `references/`
- `examples/`

## Exclude

- `extracted-models/`
- Local PDFs, scans, or book source files
- Any long source-derived excerpts that are not necessary for operating the skill

## Final Content Check

- The public package should contain operational guidance, not source-book reproduction.
- Examples should be original or user-provided, not copied from copyrighted publications.
- Review outputs should not invent line numbers, citations, tables, figures, statistics, or manuscript context.
